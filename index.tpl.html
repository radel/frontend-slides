Presenter mode

Normal view Presenter mode

In presenter mode, you can see the current and the upcoming slide.

To enter presenter mode, you just need to attach ?preview in your URL. Try it out!

Slides are synchronized between tabs.

Code highlighting

applause.directive('slide', function (Appdata) { return { restrict: 'EA', transclude: true, replace: true, scope: {} }; });

Applause uses Prism JS for code highlighting. You need to add the "language-*" class and the "prism" attribute to your element:
<div prism class="language-javascript">your code here</div>

 ##Markdown content write your slide in markdown if you like it - list item **one** - list item *two* Add the "markdown" attribute to the slide element `<slide markdown>`

Auto fitting images

Add the "bg-img" attribute to your slide element:
<slide bg-img="path/to/image">
without worrying about the image size.

You can add credits too (with autolink)
<slide bg.img="..." credits="...">

Add
class="abs-center"
to an item to center it
